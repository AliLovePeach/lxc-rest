**ALPHA**
*Do not run it in production*

## Installation
```
apt install python3-lxc
git clone --recursive https://github.com/lxc-webpanel/lxc-rest.git
cd lxc-rest
pip install -r requirements.txt
python3 install/setup.py
python3 run.py
```
